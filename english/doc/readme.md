# English idiom

## The English levels

For foreigners, the levels are the following:

English Basic User

* A1 (Beginner)

* A2 (Elementary English)

English Independent User

* B1 (Intermediate English)

* B2 (Upper-Intermediate English)

Proficient English User

* C1 (Advanced English)

* C2 (Proficiency English)
