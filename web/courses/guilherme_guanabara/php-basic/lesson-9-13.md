# 9. Conditional if

## Ternary

```php
$biggest = 1 > 0 ? 1 : 0
```

> if 1 > 0 biggest = 1, else biggest = 0

## Common if usage:

```php
if(1 > 0)
{
    ...
}
```

```php
if(1 > 0)
    ...
```

* Curly bracket is optional
